MemApp

MemApp is a small system application that I helped to develop with my team during my first year of college at PHINMA University of Pangasinan. It is an application written in Java that lets the user create their own set of flash cards for studying. It is similar to real flashcards where the users are shown a "card" for each term, which they can flip over by clicking or using the arrow keys.

Snake

Snake is a small and short game that I wrote in Python using Pygame to build my skills around the language. The game is like the classic Snake game among old mobile phones that became popular in the early 2000s. The goal of the game is to simply obtain the highest score by letting your snake crawl and eat apples to get longer.

Church Scheduler

Church Scheduler is a system application that I helped in developing with a team around my third year of college in STI College Global City. It is written in C# and uses SQLite as its database. The application is capable of creating and storing events related to church activities such as weddings, baptisms, and masses. It stores detailed information about the events such as attendees, required documents, time and date. It is also capable of searching, filtering and sorting the events that the user wishes.
